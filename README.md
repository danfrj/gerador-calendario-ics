# Gerador de Calendário (.ics)

API para geração de arquivos de calendário no formato .ics, permitindo a exportação de eventos.

##  Objetivo
Projeto educacional focado no aprendizado de manipulação de dados, arquivos e padrões de comunicação (RFC 5545) usados no mercado.

##  Tecnologias
* **Java 17**
* **Spring Boot 3.2.0**
* **Spring Web (REST API)**
* **Lombok**
